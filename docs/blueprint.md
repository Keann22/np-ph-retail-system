# **App Name**: RetailFlow

## Core Features:

- User Authentication and RBAC: Secure login system with role-based access control (Owner, Admin, Warehouse Manager, Sales) to protect sensitive data and manage user permissions.
- Product Catalog Management: Add, edit, and categorize products with details like name, SKU, description, category, supplier link, and images. Support product variations (size, color) with separate cost and selling prices.
- Inventory Control and Tracking: Track quantity on hand, quantity committed, and quantity available. Calculate quantity to order from supplier to optimize inventory levels and stock movements. Log all stock changes for auditing purposes.
- Customer Management: Store customer details, including name, contact information (phone, email, Facebook profile link), and multiple shipping addresses. Maintain a history of orders for each customer.
- Order Processing and Sales Management: Manually enter orders with support for various payment types: full payment, lay-away (Hulugan), and installment. Manage stock allocation for each order and automatically record costPriceAtSale and sellingPriceAtSale to track profit
- Payment Tracking and Reconciliation: Record all incoming payments, linked to specific orders, and support various payment methods. Automatically update amount paid and balance due on orders, creating real time summaries
- AI-Driven Recommendation Tool: The AI will use operational data to intelligently decide which marketing content and products a particular sales associate should promote based on recent order history.

## Style Guidelines:

- Primary color: Deep blue (#2E5CB8) to evoke trust, stability, and reliability, crucial for financial and operational tasks.
- Background color: Light blue-gray (#E0E5EB), offering a calm, neutral backdrop that reduces eye strain during long work sessions.
- Accent color: Vibrant orange (#E07A1B) to highlight important actions, notifications, and key financial indicators.
- Body and headline font: 'Inter' (sans-serif), providing a modern, clean, and highly readable font for all text elements.
- Code font: 'Source Code Pro' for displaying code snippets
- Use a consistent set of clean, geometric icons to represent different functions and categories within the app.
- A clear and well-organized layout that balances information density with ease of use. Key metrics should be prominently displayed in a dashboard-style view.
